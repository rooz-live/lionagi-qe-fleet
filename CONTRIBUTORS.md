## Contributors

This project follows the [All Contributors](https://allcontributors.org/) specification.
Contributions of any kind are welcome!

### How Contributions are Recognized

We recognize all types of contributions, including:

- Code contributions (features, bug fixes, tests)
- Documentation improvements
- Bug reports and feature requests
- Community support (answering questions, helping others)
- Design and UI/UX improvements
- Infrastructure and tooling
- Translation and localization
- Content creation (blog posts, tutorials, videos)

## Becoming a Contributor

We welcome contributions from everyone! Here's how to get started:

1. **Read the Guidelines**: Review [CONTRIBUTING.md](CONTRIBUTING.md) and [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md)
2. **Find an Issue**: Look for issues labeled `good first issue` or `help wanted`
3. **Ask Questions**: Don't hesitate to ask for help in GitHub Discussions
4. **Make a Contribution**: Submit a pull request or help in other ways
5. **Get Recognized**: You'll be added to this list!

## How to Add Yourself

When your PR is merged, you can add yourself to the contributors list using the
[All Contributors bot](https://allcontributors.org/docs/en/bot/overview):

```
@all-contributors please add @username for code, tests, docs
```

Or a maintainer will add you during the PR review process.
